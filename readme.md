# StartupUTC

Outil de visualisation du tissu entrepreunarial de l'UTC. #IC05

## Getting started

Installer les dépendances
```
composer install
npm install
```

Compiler le projet
```
gulp
```

## Visualisation sous Gephi

Générer les fichiers `edges.csv` et `nodes.csv` avec la commande:
```console
php artisan similarities:calc
```
